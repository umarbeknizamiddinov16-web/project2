# 🌌 Premium SaaS User Dashboard

Защищенная панель управления (Личный кабинет) со встроенной аутентификацией пользователей и изящным темным дизайном.

## 🚀 Технологический стек
- **Framework:** Next.js 16 (App Router)
- **Authentication:** Clerk Auth (полный контроль сессий и защиты путей)
- **Routing:** Next.js 16 Proxy Architecture (`src/proxy.js`)
- **Styling:** Tailwind CSS v4
- **Database:** Neon PostgreSQL (подготовлен пул подключений)

## 💎 Реализованные фичи и решенные проблемы
1. **Премиальный Dark UI:** Интерфейс полностью переверстан с нуля на базе Tailwind v4 с интерактивным сайдбаром, карточками аналитики безопасности и Verified-статусами.
2. **Next.js 16 Proxy:** Успешно внедрена новейшая конвенция `proxy.js` вместо устаревших middleware для корректного проброса авторизационных куков Clerk.
3. **Исправление петель редиректа:** Устранена проблема зацикливания маршрутов на `localhost` путем синхронизации переменных окружения `.env.local` и сессионных токенов.
